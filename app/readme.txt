# Pokedex3

Pokedex3 es una aplicación Android que permite a los usuarios explorar una lista de Pokémon, ver detalles sobre cada Pokémon y ordenar la lista de Pokémon por altura.

## Características

- Lista de Pokémon: Muestra una lista de Pokémon obtenida de la API de PokeAPI.
- Detalles del Pokémon: Al hacer clic en un Pokémon en la lista, se muestra una pantalla de detalles con más información sobre el Pokémon, incluyendo su nombre, imagen, habilidades, color  y textos de sabor.
- Lista de Habilidades: Muestra una lista de habilidades obtenida de la API de PokeAPI.
- Detalles de la habilidad: Al hacer clic en una habilidad en la lista, se muestra una pantalla de detalles con más información sobre la habilidad, incluyendo su nombre, descripción, efectos y una lista de Pokémon que la usan.

## Instalación

Para instalar Pokedex3, sigue estos pasos:

1. Clona el repositorio de Pokedex3 en tu máquina local.
2. Abre el proyecto en Android Studio.
3. Ejecuta el proyecto en un emulador o dispositivo Android.

## Uso

Para usar Pokedex3, sigue estos pasos:

1. Abre la aplicación en tu dispositivo Android.
2. Verás el menú inicial con dos botones: "Pokémon List" y "Ability List", que te permtirán acceder a las listas de Pokémon y habilidades, respectivamente.
3. En la "Pokémon List" puedes desplazarte por la lista para ver más Pokémon. Al hacer clic en un Pokémon, se mostrará una pantalla de detalles con más información sobre el Pokémon.
4. En la "Ability List" puedes desplazarte por la lista para ver más habilidades. Al hacer clic en una habilidad, se mostrará una pantalla de detalles con más información sobre la habilidad.
5. Puedes navegar atrás para volver a acceder al menú inicial.
